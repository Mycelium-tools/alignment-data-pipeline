To start, please generate a list of {count} different types of documents that could appear in this corpus (for example, a blog post or a podcast transcript). Later steps will split each type into several subtypes and then write the actual documents, so this list determines the diversity of the whole corpus. Each type should be a kind of document that genuinely appears on the internet, and the list should collectively cover a wide range of formats and contexts.

For each type, provide:
- type_name: a concise label for the document type (for example, "blog post")
- description: 1-2 sentences describing this type of document

Return your answer as a JSON array with {count} objects, each with keys: type_name, description. Return only the JSON array, no preamble or explanation.
